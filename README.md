# Diffy - Mobile Robot

A differential drive mobile robot project focused on design, development, simulation and real-world testing.

## Objective
To design and build a stable, efficient mobile robot platform for autonomous and manual control experiments.

## Repository Structure

Design_files/       -> CAD models, STEP/STL/DXF, renders  
Simulation_data/    -> Simulation results, plots, analysis  
Documentation/      -> Reports, BOM, drawings and references  

## Tools & Technologies
- SolidWorks / Fusion 360
- Arduino / Microcontrollers
- Differential drive kinematics
- FDM 3D printing
- Sensors & motor drivers
